Fixes several old bugs in game files.

Modifies simulation - all players in multiplayer must have this mod.

Built for patch 1.45 (Eivonn's fine) , 2025-10-06+

# Fixes
- advent / radiance battleship / animosity : being negatively affected by psi power
- vasari alliance / sularkon tyrant / subversion grid : incorrect tooltip
- advent reborn / temple of purification : being buildable multiple times
- advent reborn / justicia dreadnought / reinvigorate / malefic locus : reduces allies' damage instead of increasing it
